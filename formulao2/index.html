Slideshow

Slideshow

  • Pages > Main Slideshow > Add Media
  • Click the "upload media" tab.
  • Upload imagery
  • Drag and drop to reorder
  • Be sure to set the media scope to "uploaded to this page"

Add captioning and colors to fields at the bottom of the Main Slideshow screen.

Features

Features

Adding, Editing and Reordering

  1. Select "features" from the side nav
  2. "Add new" to create a new one.
  3. Click title to edit existing feature
  4. Select Re-order to drag and drop the arrangement

Feature Content Editing

Feature Content Editing

  1. Add title
  2. Create the text content
  3. Set the featured image
  4. Optional secondary title
  5. Choose the alignment of the feature
  6. Select "super large" if you want this feature to be Parallax'd
  7. Update the background ratio if you wish to adjust the motion settings
  8. Choose color of the header text
